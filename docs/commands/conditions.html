Δομή ελέγχου

οι δομές ελέγχου χρησιμοποιούνται για τη λήψη αποφάσεων μέσα σε ένα πρόγραμμα. Αυτές οι δομές επιτρέπουν τον έλεγχο της ροής του προγράμματος, ανάλογα με το αν οι συνθήκες είναι αληθείς ή ψευδείς.
Οι δομές ελέγχου είναι:

1) if: Ελέγχει μια συνθήκη και εκτελεί έναν κώδικα αν η συνθήκη είναι αληθής.

x = 10
if x > 5:
    print("Το x είναι μεγαλύτερο από 5")

2) else: Εκτελεί έναν κώδικα αν η συνθήκη στο if ή στο elseif (αν υπάρχει) είναι ψευδή.

x = 3
if x > 5:
    print("Το x είναι μεγαλύτερο από 5")
else:
    print("Το x είναι μικρότερο ή ίσο με 5")

3) elif: Προσθέτει επιπλέον συνθήκες ελέγχου σε μια δήλωση if.

x = 7
if x > 10:
    print("Το x είναι μεγαλύτερο από 10")
elif x == 7:
    print("Το x είναι ίσο με 7")
else:
    print("Το x είναι μικρότερο από 10 αλλά δεν είναι 7")

Λογικοί Τελεστές

Μπορείς να συνδυάσεις πολλαπλές συνθήκες με τους λογικούς τελεστές:

  1. and: Αληθές μόνο αν όλες οι συνθήκες είναι αληθείς.
  2. or: Αληθές αν τουλάχιστον μία συνθήκη είναι αληθής.
  3. not: Αντιστρέφει την αλήθεια της συνθήκης.
