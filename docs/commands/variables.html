Τύποι δεδομένων

Για να χρησιμοποιήσουμε μια μεταβλητή, χρειαζόμαστε να της δώσουμε ένα όνομα και στην συνέχεια να της εκχωρήσουμε κάποια τιμή. Με το σύμβολο = δίνουμε μια τι στην μεταβλητή.
Για παράδειγμα 

x=15

Προσοχή, δεν δηλώνουμε τον τύπο δεδομένων που θα έχει η μεταβλητή, όπως κάνουμε σε άλλες γλώσσες προγραμματισμού όπως η C.

Τύποι Δεδομένων

Στην Python οι τύποι δεδομένων είναι:

  • οι αριθμοί: int , float , complex (μιγαδικοί αριθμοί)
  • Συμβολοσειρές: str
  • Ο λογικός τύπος: bool (έχει 2 τιμές true και false)
