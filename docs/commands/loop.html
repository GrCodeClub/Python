Δομή επανάληψης

Η δομή επανάληψης (ή βρόχος, loop στα Αγγλικά) είναι ένας τρόπος να εκτελούνται οι ίδιες εντολές πολλές φορές μέχρι να ικανοποιηθεί μια συγκεκριμένη συνθήκη. Αυτές οι δομές χρησιμοποιούνται για να επαναλάβουν ένα σύνολο εντολών χωρίς να χρειάζεται να τις γράφουμε πολλές φορές. Οι πιο κοινές δομές επανάληψης είναι:

  1. for: Χρησιμοποιείται για επανάληψη σε μια ακολουθία (λίστες, συμβολοσειρές, range κ.λπ.).
  2. while: Επαναλαμβάνει τον κώδικα όσο μια συνθήκη είναι αληθής.

For

Η for επανάληψη χρησιμοποιείται για να επαναλάβεις κώδικα για κάθε στοιχείο μιας ακολουθίας.

Χρήση range() με το for loop. Η συνάρτηση range() δημιουργεί ακολουθίες αριθμών

# Εκτύπωση αριθμών από 0 έως 4
for i in range(5):
    print(i)

Το range() μπορεί επίσης να δεχτεί αρχική τιμή, τελική τιμή, και βήμα:

# Εκτύπωση ζυγών αριθμών από 2 έως 10
for i in range(2, 11, 2):
    print(i)

while επανάληψη

Η while επανάληψη εκτελεί τον κώδικα όσο η συνθήκη που ελέγχεται είναι αληθής. Αν η συνθήκη γίνει ψευδής, η επανάληψη σταματά.

# Εκτύπωση αριθμών από 1 έως 5
i = 1
while i <= 5:
    print(i)
    i = i + 1

Χρήση του break και continue

Η εντολή break χρησιμοποιείται για να διακόψει την επανάληψη και να βγει από το loop πριν η συνθήκη γίνει ψευδής.

for i in range(10):
    if i == 5:
        break  # Διακοπή όταν i είναι 5
    print(i)

Η εντολή continue χρησιμοποιείται για να παραλείψει το υπόλοιπο των εντολών μέσα στο loop και να προχωρήσει στην επόμενη επανάληψη.

for i in range(5):
    if i == 3:
        continue  # Παράλειψη όταν i είναι 3
    print(i)
