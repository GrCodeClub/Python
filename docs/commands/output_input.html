Είσοδος / Έξοδος

Έξοδος (Output)

print() Χρησιμοποιείται για να εμφανίσει δεδομένα στην κονσόλα.

print("GrCode Club")

Μπορείς να εμφανίσεις πολλαπλές τιμές ή μεταβλητές με διαχωρισμό από κόμμα:

name = "Νίκος"
age = 25
print("Το όνομα μου είναι", name, "και είμαι", age, "χρονών.")

Είσοδος (Input)

Η είσοδος από το χρήστη γίνεται κυρίως με τη χρήση με την εντολή input()

input(): Χρησιμοποιείται για να λαμβάνεις δεδομένα από τον χρήστη με τη μορφή κειμένου (string).

name = input("Ποιο είναι το όνομά σου; ")
print("Γειά σου, " + name)

Η τιμή που επιστρέφεται από την input() είναι πάντα string, οπότε αν χρειαστείς κάποια άλλη μορφή (π.χ., αριθμό), θα πρέπει να τη μετατρέψεις:

age = int(input("Πόσο χρονών είσαι; "))
print("Είσαι " + str(age) + " χρονών.")
