Requests

Εγκατάσταση

pip install requests

Εισαγωγή

import requests

GET Αίτημα

response = requests.get('https://api.github.com')
print(response.status_code)  # Κατάσταση HTTP (π.χ., 200, 404)
print(response.text)         # Περιεχόμενο της απάντησης ως string

POST Αίτημα

url = 'https://httpbin.org/post'
data = {'key': 'value'}
response = requests.post(url, data=data)
print(response.json())  # Επιστρέφει το JSON περιεχόμενο της απάντησης

Παράμετροι GET Αιτήματος

params = {'param1': 'value1', 'param2': 'value2'}
response = requests.get('https://httpbin.org/get', params=params)
print(response.json())

Δεδομένα POST Αιτήματος ως JSON

import json
url = 'https://httpbin.org/post'
headers = {'Content-Type': 'application/json'}
json_data = {'key': 'value'}
response = requests.post(url, headers=headers, data=json.dumps(json_data))
print(response.json())

Βασική Αυθεντικοποίηση

Αποστέλλει αίτημα με όνομα χρήστη και κωδικό.

from requests.auth import HTTPBasicAuth
url = 'https://httpbin.org/basic-auth/user/pass'
response = requests.get(url, auth=HTTPBasicAuth('user', 'pass'))
print(response.json())

Λήψη Αρχείων

Κατεβάζει ένα αρχείο από έναν URL

url = 'https://www.example.com/somefile.zip'
response = requests.get(url)
with open('somefile.zip', 'wb') as file:
    file.write(response.content)

Χρόνος Αναμονής (Timeout)

Ορίζει το μέγιστο χρόνο αναμονής για μια αίτηση

response = requests.get('https://api.github.com', timeout=5)  # 5 δευτερόλεπτα

Request and Response Objects

response = requests.get('https://httpbin.org/get')
print(response.request.method)  # Μέθοδος αιτήματος (GET)
print(response.status_code)     # Κατάσταση HTTP
print(response.headers)         # Headers της απάντησης
print(response.text)            # Περιεχόμενο της απάντησης

SSL Cert Verification

Η requests βιβλιοθήκη ελέγχει αυτόματα τα SSL certificates για την ασφάλεια των συνδέσεων. Μπορείς να απενεργοποιήσεις την επαλήθευση, αν χρειαστεί (αν και αυτό δεν συνιστάται για παραγωγικό περιβάλλον)

response = requests.get('https://self-signed.badssl.com/', verify=False)

Keep-Alive

Η requests βιβλιοθήκη υποστηρίζει την σύνδεση Keep-Alive για να διατηρεί ανοιχτές συνδέσεις HTTP για πολλαπλά αιτήματα.

# Χρησιμοποιώντας Session για Keep-Alive
session = requests.Session()
response = session.get('https://httpbin.org/get')

Streaming Uploads

with open('large_file.zip', 'rb') as f:
    response = requests.post('https://httpbin.org/post', data=f)
